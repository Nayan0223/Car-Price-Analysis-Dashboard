# Car Price Analysis Dashboard

## Overview
Analyzed used car data using Python and Tableau to identify pricing trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Tableau

## Features
- Fuel Type Analysis
- Company-wise Price Analysis
- Year vs Price Analysis
- KM Driven Analysis
## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Tableau Dashboard
- Fuel Type Analysis
- Company-wise Price Analysis
- Year-wise Price Trends
- Kilometer Driven Analysis
- Car Price Prediction using Machine Learning
- Model Evaluation using R² Score, MAE, and RMSE
